Stockify


Stockify is a stock market trading application that allows for its users to practice trading stocks risk free. Prices, graphs, and new articles for every stock listed on U.S. exchanges are updated in real-time, allowing for users to experience the fast-paced price action that markets consist of.


Built with: MongoDB, Express, React, Node, Javascript, HTML, CSS, Plotly, IEX API, and Alphavantage API

Puppy Love


Puppy Love is an application that matches prospective dog owners to dogs that are currently in animal shelters within 20 miles. The user takes a brief survey and based on their answers, an algorithm will sort through the largest animal shelter database in the world and return the 5 best matches. This application won 3rd place in General Assembly's 24 hour "Hindsight 2020" hackathon.


Built with: Javascript, HTML, CSS, React, Bootstrap, and Petfinder API


Github Repository
